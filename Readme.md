# UI Reading Space Enhancer (Chrome Extension)

A lightweight Chrome extension that improves the reading experience by maximizing usable screen space.

The extension hides distracting UI elements (header and input/search area) and brings them back automatically on hover, allowing users to focus on content without permanently removing controls.

---

## Features

- Hides header and input/search UI
- Restores hidden elements on hover
- Maximizes vertical reading space
- Lightweight and fast
- No permanent UI changes
- No data collection

## How It Works

- Injects minimal CSS and JavaScript into the page
- Dynamically hides selected UI components
- Uses hover detection to restore UI when needed
- Runs locally in the browser

---

## Installation (Developer Mode)

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder

---

## Use Case

This extension was built as a **proof of concept** to demonstrate how small UI changes can significantly improve readability in content-heavy interfaces.
